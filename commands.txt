API Endpoints Explained: The Complete Guide
WHAT are Endpoints?
An API endpoint is a specific URL where your application can send requests to interact with a server. Think of it as a door to your application - each door serves a specific purpose.

In your chatbot:
/ask - The main door for asking questions
/feedback - Door for sending feedback
/health - Door to check if the system is working
